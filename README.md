# Sistem Pengesanan Banjir 🚨

## Pengenalan
Sistem ini dibangunkan menggunakan Arduino untuk mengesan paras air sebagai langkah awal amaran banjir. Sistem akan memberikan amaran melalui LED dan buzzer serta menggerakkan servo apabila paras air mencapai tahap bahaya.

## Objektif
1. Mengesan paras air secara automatik  
2. Memberikan amaran awal banjir  
3. Mengawal pergerakan servo berdasarkan keadaan air  
4. Meningkatkan keselamatan kawasan persekitaran  

## Pengisian
Sistem ini menggunakan sensor (potentiometer dalam simulasi Tinkercad) untuk membaca paras air. Apabila nilai melebihi had yang ditetapkan:
- LED akan menyala sebagai amaran visual  
- Buzzer akan berbunyi sebagai amaran bunyi  
- Servo akan bergerak sebagai tindak balas sistem  

Apabila paras air kembali normal, semua komponen akan kembali ke keadaan asal.

## Komponen
- 1 unit Papan Mikropengawal Arduino Uno  
- 1 unit Breadboard  
- 1 unit LED Merah  
- 1 unit Buzzer  
- 1 unit Motor Servo SG90  
- 1 unit Perintang 220 ohm  
- 1 unit Potentiometer (menggantikan Water Level Sensor dalam Tinkercad)  
- 15 unit Wayar  

## Kesimpulan
Sistem ini dapat membantu memberikan amaran awal banjir dengan cepat dan berkesan serta sesuai digunakan dalam projek pembelajaran dan pertandingan.
